Project 1: Object Detection and Tracking

In a collaborative project, my class partner and I were tasked to write a program that would take in a video and count the number of people that enter the camera frame. We were given a file that registered and updated the different IDs for each person. Using that file we passed each frame of the video sequentially through the yolo model to identify people before updating the IDs using the given sort.py file. We then added a display on the output video to show how many people had been in the frame of the given camera shot.

We also created an image detector that takes an image and draws bounding boxes labeling each class found in the image along with its confidence score.

Project 2: Path Finding

We created a maze solver that takes in a maze file and finds the shortest path from the start to end points using either dijkstra's or the A-star algorithms. We also created a display to show how the maze is solved using the specified algorithm. The below video shows the program solving a maze using the A-star algorithm.

Project 3: Path Tracking and Object Removal

We created our own tracking method based on the centroid of the bounding boxes in each frame and used it to keep track of people within the video and trace the path they have traveled.

We also created an object removal program that finds and removes a specified class for an image and replaces it with a specified chunk to the right of the object. Below shows the removal of people from a picture of a beach.

Project 3: Frogger Game

I created an interactive game in C++ based on Frogger as a volunteer project per the request of a middle school teacher in my community. The goal of the game is to swap the X and O pieces from one side to the other in as few moves as possible. Pieces can move to an empty space next to them or hop over a piece to an empty space.
