Project 2: Path Tracking and Object Removal

Focus: MOTS with YOLO object detection

Using video input, track people's positions as they move. Then output a video displaying the path people have traveled.

We created our own tracking method based on the centroid of the bounding boxes in each frame and used it to keep track of people within the video and trace the path they have traveled.

We also created an object removal program that finds and removes a specified class for an image and replaces it with a specified chunk to the right of the object. Below shows the removal of people from a picture of a beach.

Project 5: Frogger Game

Goal:
Create a coding example which demonstrates that coding can be simple and intuitive to read & write. Coding is something anyone can do!

Intended Audience
Middle School Math Students (12-15 year old)

Math Level: Pre-Algebra

Programming Background: None

Criteria
- Intuitive enough for a non-programmer to read and be able to understand what the code does
- Fun (a game similar to one of their in-class math exercises)

The goal of the game is to swap the X and O pieces from one side to the other in as few moves as possible. Pieces can move to an empty space next to them or hop over a piece to an empty space.
